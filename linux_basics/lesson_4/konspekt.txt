< - перенаправление стандартного потока ввода для чтения из процесса или файла

> - перенаправление потока вывода для записи в файл или процесс. Эта команда 
перезаписывает данные в файлах для записи новых данных в конец файла 
используется оператор >> (он не перезаписывает содержимое).

2> и 2>> перенаправление стандартного потока ошибок, например 2>файл. 
синтаксис аналогичен операторам потока вывода.

==============================================================================

PID/PPID - (process identifier) уникальный номер (идентификатор) процесса 
           (parent process identifier) идентификатор родительского процесса. ОС 
           взаимодействует с процессами через их идентификаторы. Главный процесс, 
           который запускает остальные имеет идентификатор - 1.

UID      - владелец процесса, пользователь, от имени которого он запущен.

CMD      - команда, запустившая япроцесс.

systemctl- утилита, через которую осуществляется управление процессами и 
           службами. Через нее также можно получать информацию о состоянии
           системы и запущеннслужбах. Более подробную информацию о работе с 
           утилитой можно получить в руководстве man systsmctl.

systemctl [start | stop | reload | status | restart] service_name - позволяет
          запустить, остановить, перечитать конфиг файл, получить инфу о службе
          и перезапустить службу (соответственно).

systemctl [enable | disable] - позволяет добавить или убрать службу из 
                               автозагрузки.

kill     - команда для завершения процессов. Полный список параметров доступен
           команде kill -l. Работает с процессами через их PID / PPID.

==============================================================================

grep     - утилита, осуществляющая поиск по строкам согласно заданного шаблона

ps       - показывает список запущенных процессов в ОС. В сочетании с grep 
           может найти и получить информацию о процессе.

top      - (table of process) выводит списокпроцессов и информацию о них, общую
           нагрузку системы. По умолчанию соритрует список процессов по нагрузке 
           на процессор в режиме реального времени. 

htop     - улучшенная версия top.

df       - показывает размер, занятое и свободное пространство на смонтированных
           файловых системах ОС.
